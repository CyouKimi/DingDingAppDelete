# DingDingAppDelete
## 本程序所用的接口仅限企业调用，ISV无法调用
由于钉钉没有提供删除微应用和E应用的按钮，查找文档后发现需要使用工具构建请求，使用本工具即可快速删除不需要的应用。注意：应用删除可能会造成损失，请谨慎操作！
